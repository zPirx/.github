# AI Policy

## How I use AI in this project

The code here is AI-drafted but human-directed and human-verified. I design the approach and run every step manually. I read every output and test every result.

No autonomous agent tools like Claude Code or Codex are used. The AI assists but does not act on its own.

My working method and verification routine are documented in the templates here: https://github.com/zPirx/.github/tree/main/templates

I am not a native English speaker and I use AI to help me write clearly. The technical decisions and the responsibility for them are mine.

## If you want to contribute

You are welcome to use AI too. I only ask that you:

- understand your change and can explain it if I ask
- disclose that AI was involved and roughly how
- test it before opening the PR

I review and merge based on whether the change is correct and fits the project. The final decision is mine but I am friendly. So ask if anything is unclear.
